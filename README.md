# Проект для раздачи статичных файлов через Nginx в Docker

## Описание

Этот проект настроен для раздачи статики (например, изображений) через Nginx в Docker контейнере. Он включает настройку SSH и конфигурацию Nginx для раздачи файлов через путь `/images`.

## Требования

- Docker
- Docker Compose (опционально)

## Структура проекта

- **`anc/`**: Папка с файлами для конфигурации Ansible.
- **`Dockerfile`**: Файл для создания Docker образа.
- **`.dockerignore`**: Файл для игнорирования ненужных файлов в контейнере.

## Настройка и запуск

1. **Клонируйте репозиторий**:
   ```bash
   git clone https://your-repository-url.git
   cd project-directory
   ```

2. **Настройка SSH ключей**:
   - Для использования SSH подключения необходимо добавить свой **публичный SSH ключ** в папку **`anc/roles/create_users/files/`** в файл **`id_rsa.pub`**.
   - Если ключи не были добавлены, Ansible не сможет сконфигурировать подключение.

3. **Создание контейнера**:
   Для сборки и запуска контейнера используйте команду:
   ```bash
   docker-compose up --build
   ```

4. **Запуск Ansible**:
   Для запуска Ansible плейбука и настройки сервера выполните:
   ```bash
   ansible-playbook -i anc/inventory.yml anc/playbook.yml
   ```

5. **Доступ к статике**:
   После запуска Nginx, статика будет доступна по пути `/images`. Просто зайдите в браузер по адресу **`http://localhost/images`**, чтобы увидеть список файлов, и **`http://localhost/images/filename`**, чтобы открыть изображение.

## Примечания

- Если вам нужно обновить публичные SSH ключи, просто поместите свой новый **публичный ключ** в файл **`anc/roles/create_users/files/id_rsa.pub`**.
- В конфигурации контейнера используется стандартный SSH порт **22**.
